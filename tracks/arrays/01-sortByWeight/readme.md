# sortByWeight

Потрібно реалізувати функцію `sortByWeight`, яка приймає массив чисел та сортує по сумі кожного елементу

**Приклад:**

```js
sortByWeight([
  3, 
  1, 
  '55', 
  33,
  '20',
]);
```

Поверне об'єкт:
```js
[
  1,
  3, 
  '20',
  33,
  '55'
]
```
